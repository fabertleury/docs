## 1. Obtenha sua API Key

Acesse o [Painel de Parceiro](https://emitezap.com.br/painel/dashboard) e gere sua chave de API na seção **Configurações > API Keys**.

Sua chave terá o formato:

```
ez_live_abc123_sk_7f8a9b2c4d6e1f3a5b7c9d0e2f4a6b8c
```

<Warning>
  Guarde sua chave em local seguro. Ela não será exibida novamente após a criação.
</Warning>

## 2. Verifique a conexão

Teste sua chave listando suas empresas cadastradas:

```bash
curl -X GET https://emitezap.com.br/api/partner/v1/companies \
  -H "X-API-Key: ez_live_abc123_sk_7f8a9b2c4d6e1f3a5b7c9d0e2f4a6b8c" \
  -H "Content-Type: application/json"
```

Se a chave estiver correta, você receberá uma lista (possivelmente vazia) de empresas.

## 3. Cadastre uma empresa

Cadastre a empresa que emitirá as notas:

```bash
curl -X POST https://emitezap.com.br/api/partner/v1/companies \
  -H "X-API-Key: ez_live_abc123_sk_7f8a9b2c4d6e1f3a5b7c9d0e2f4a6b8c" \
  -H "Content-Type: application/json" \
  -d '{
    "cnpj": "12.345.678/0001-90",
    "razao_social": "Tech Solutions Ltda",
    "nome_fantasia": "TechSol",
    "regime_tributario": "simples_nacional",
    "municipio": "São Paulo",
    "uf": "SP",
    "email": "fiscal@techsol.com.br",
    "inscricao_municipal": "12345678"
  }'
```

Anote o `id` retornado — você precisará dele para emitir notas.

## 4. Emita sua primeira NFS-e

Com a empresa cadastrada e créditos disponíveis, emita uma nota fiscal de serviço:

```bash
curl -X POST https://emitezap.com.br/api/partner/v1/emit \
  -H "X-API-Key: ez_live_abc123_sk_7f8a9b2c4d6e1f3a5b7c9d0e2f4a6b8c" \
  -H "Content-Type: application/json" \
  -d '{
    "company_id": "comp_9f8a7b6c5d4e3f2a",
    "servico": {
      "descricao": "Desenvolvimento de software sob demanda",
      "valor": 5000.00
    },
    "tomador": {
      "cpf_cnpj": "987.654.321-00",
      "razao_social": "Maria Fernanda Silva",
      "email": "maria@example.com"
    },
    "metadata": {
      "external_id": "pedido-001"
    }
  }'
```

Resposta:

```json
{
  "invoice_id": "inv_a1b2c3d4e5f6",
  "status": "processing",
  "external_id": "pedido-001",
  "created_at": "2026-04-15T14:30:00Z"
}
```

<Note>
  A emissão é assíncrona. O status inicial será `processing`. Configure um [webhook](/conceitos/webhooks) para ser notificado quando a nota for autorizada.
</Note>

## 5. Consulte o status

Acompanhe o status da nota emitida:

```bash
curl -X GET https://emitezap.com.br/api/partner/v1/invoices/inv_a1b2c3d4e5f6 \
  -H "X-API-Key: ez_live_abc123_sk_7f8a9b2c4d6e1f3a5b7c9d0e2f4a6b8c"
```
