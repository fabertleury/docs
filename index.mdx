# EmiteZap Partner API

A Partner API permite que resellers e integradores emitam notas fiscais (NFS-e e NF-e) para seus clientes de forma programatica.

## Como funciona

1. **Cadastre-se** como parceiro no EmiteZap
2. **Gere uma API Key** no painel em API Keys
3. **Cadastre a empresa** do seu cliente via API
4. **Emita notas** enviando os dados do servico e tomador
5. **Receba webhooks** com o status da nota (autorizada, rejeitada, etc.)

## Base URL

```
https://emitezap.com.br/api/partner/v1
```

## Autenticacao

Todas as requisicoes devem incluir o header `X-API-Key`:

```bash
curl -X GET https://emitezap.com.br/api/partner/v1/credits \
  -H "X-API-Key: ez_live_abc12def_x9k2mNpQrLvT8wY3jH6cB1nA4sUeI0oZ"
```
